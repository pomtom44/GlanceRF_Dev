Welcome to GlanceRF

GlanceRF is a modular ham-radio dashboard: a live grid of tiles for APRS, maps, GPS, and more—so you can see what matters at a glance on your display.

We’ll walk you through a few setup steps—hardware, your station profile, screen layout, and tips—so your profile is configured and you’re ready to use the app.

Menu

Right-click to open the menu. From the menu you can go to Setup, Layout & Config editor, Modules list, or run Manual Updates. (On the main dashboard you can also press M.)

GPS module

Add the GPS module to the grid to see live GPS stats (position, time, altitude, speed, track, satellites) when a GPS receiver is connected.

Map only modules

Use Map only modules to add modules that feed data to the map overlay (e.g. satellites, APRS) without taking a grid cell on the main dashboard.

Setup guides

Check the GlanceRF website for setup guides (quick start, GPIO wiring, and more).
